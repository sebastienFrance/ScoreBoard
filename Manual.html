Score Log Manual

All modifications you are performing are saved in the real time.

It is used to add a new player in the game. You can add a new player at any time during a game.

It is used to modify the player list. When you click this button you can reorder the list of players and/or delete a player from the game. Tips: To delete a player you just need to swipe your finger from right/left to left/right on the player row and a delete button will appear.

To add a score to a player you just need to touch the player row. It will open a new window to add a new score, it can be also used to remove a score in can of mistake or to display score details of the player.

You can choose to end the current game and return to the main meny or to create a new game with the same player. In any cases the current game is still saved in your iPhone and you can reselect it using from the main menu with "Continue a game". No data is deleted except if you really request it.

It is used to open the options window to configure your game

It is used to send the current game result to your friends by mail. You can use this option at any time during the game and how many times you want

It is used to reorder the player list based on their ranking.

Score Log main features:

  • Create your players from your address book or manually
  • Take a snapshot of your players
  • Select if highest or lower score wins
  • Games are automatically saved on each modification
  • Continue or delete an old game
  • Send an email to your friends with the score result
  • Manage positive and negative Score
  • Cancel easily a score error and everything is automatically recomputed
  • Add or remove a player during a game
  • Create a new game with same players in one touch
  • ...
